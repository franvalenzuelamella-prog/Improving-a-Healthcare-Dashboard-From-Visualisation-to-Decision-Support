# Improving-a-Healthcare-Dashboard-From-Visualisation-to-Decision-Support
Improving a healthcare dashboard by enhancing clarity, usability, and decision-making through better visual design and analytics.

## Healthcare Dashboard Redesign (Power BI)

## Overview
This project presents the evaluation and redesign of a healthcare dashboard focused on COPD prevalence and patient registration data. The original dashboard was developed as part of a group project. This work improves its usability, clarity, and effectiveness as a decision-support tool.

## Objective
To enhance the dashboard by improving:
- Visual hierarchy
- Readability
- Analytical clarity
- User guidance

## Key Improvements
- Replaced gauge visuals with KPI cards to improve readability
- Introduced clear visual hierarchy (primary KPI + supporting metrics)
- Reduced visual clutter by simplifying chart types
- Applied filtering (Top practices) to improve interpretability
- Repositioned filters for better usability
- Added “Key Insights” section to guide interpretation

## Results
The redesigned dashboard allows faster understanding of:
- Variation in COPD prevalence across practices
- Relationship between clinic size and patient registration
- Trends in prevalence over time
