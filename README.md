# Improving-a-Healthcare-Dashboard-From-Visualisation-to-Decision-Support
Improving a healthcare dashboard by enhancing clarity, usability, and decision-making through better visual design and analytics.

## Overview
This project focuses on improving the effectiveness of a healthcare dashboard built in Power BI. The goal was not only to visualise data, but to enhance clarity, usability, and decision-making support.

The original dashboard contained multiple visual elements and metrics related to patient profiles, symptoms, and activity. While visually rich, it presented challenges in interpretation and prioritisation of information.

This project documents the process of analysing, critiquing, and redesigning the dashboard to improve its analytical value.

This project uses synthetic/anonymised data for demonstration purposes only.
